# alkahest-retort

The apparatus. Tubes flowing into each other. The mad tangled workbench.

A substrate for thermodynamic knowledge systems — graph primitives, phase physics, observer frames. Not the work. The vessel where transformation happens.

## What lives here

- **Graph** — nodes, edges, phases. No opinions about what they represent.
- **Thermal** — Fourier diffusion over graphs. Heat sources, conductance, cooling.
- **Gas** — PV=nRT over any measurable system. Pressure, volume, amount, temperature.
- **Frames** — observer positions, confidence, parallax. The math, not the CLI.

## What builds on it

- [`alkahest-ts`](https://github.com/Unity-Environmental-University/alkahest-ts) — TypeScript type dissolution, Otter loop
- [`rhizome-alkahest`](https://github.com/Unity-Environmental-University/rhizome-alkahest) — positioned human knowledge graph, edge CLI

## The name

A retort is the glass vessel where distillation happens — curved neck, sealed, everything flowing through it and back. It is also a response. The thing that talks back to what went in.

Alkahest dissolves everything into first matter so it can become something else.
The retort is where that happens.
